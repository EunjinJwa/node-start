## Module
모듈은 javascript의 라이브러리와 같다.
모둘을 추가하기 위해서는 `require()` function을 모듈이름과 함께 사용하면 된다.
```
var http = require('http')
```
node의 내장 모듈 참조 : [[Built-in Modules]](https://www.w3schools.com/nodejs/ref_modules.asp)
